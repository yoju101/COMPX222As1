What is HTTP and HTTPS?

Both HTTP and HTTPS are protocols that the internet uses to label the way information is transported from one location of the internet to another. Hypertext Transfer Protocol (HTTP) is the underlying network protocol that is used to request hypertext documents when the user follows a hyperlink.

HTTP assumes a client/server network architecture in which the web browser is the client and the web host is the server. Note that roles may change if your computer is both a web browser and a web host. One of the big differences between these two is that one HTTPS which stands for “Hypertext Transfer Protocol Secure” is secure and HTTP stands for “Hypertext Transfer Protocol” and is not.
