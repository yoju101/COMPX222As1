What is a request?

What is the format of a HTTP request?

The first line is the request line, is holds the command like for example GET PUT or POST

The second line is the request header fields, these headers are not visible to the user and can only be logged by the server and applications used.

Then we have an empty line to show the seperation between the header and the content the content is the last line which is the optional request body which is the data being sent to the client however it is not always necessary to send a resonse body.

Here we have a list of some of commen HTTP request error codes that can be found.

  • Moved Permanently - 301
  • Bad Request - 400
  • Forbidden - 403
  • Not Found - 404
  • Server Error - 500
